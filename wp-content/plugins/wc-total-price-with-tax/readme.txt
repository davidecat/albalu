A user-friendly plugin to calculate and display the total price, including taxes, for products, shipping, and fees on WooCommerce admin orders

== Description ==
This amazing and simple plugin adds a column to the WooCommerce Orders admin page, displaying the Total Price, including taxes, for each row. It seamlessly calculates the Total Price for various rows, whether they represent products, shipping, or fees.



= Tags =
WC, woocommerce, orders, taxes, total price, shipping, fee.

= How to use this plugin? =
Just install the plugin, and it will automatically add a column to the admin 'Edit Order' page, displaying the Total Price, including taxes.

 
== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. There will now be displayed the extra column.
